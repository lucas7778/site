BkdEda

Módulo responsável pela análise exploratória dos dados

Este módulo recebe um DataFrame e realiza uma análise exploratória, retornando as principais características dos dados em questão.

Apresenta as seguintes classes:

BkdEda

class BkdEda.__init__(self, bkd_dados)

Construtor da classe BkdEda

  • :param bkd_dados: DataFrame sobre o qual serão feitos as análises
  • :type bkd_dados: pandas.DataFrame
class BkdEda.bkd_dados(self)

Possibilita acesso ao atributo bkd_dados

  • :return: dados lidos e armazenados
  • :rtype: pandas.DataFrame
class BkdEda.bkd_type(self)

Checa os tipos dos dados fornecidos

  • :return: tipo de dados para cada variável
  • :rtype: pandas.Series
class BkdEda.bkd_nuniqnan(self)

Conta o número de dados únicos por variável, considerando valores NaN

  • :return: numero de valores únicos por variável
  • :rtype: pandas.DataFrame ou pandas.Series
class BkdEda.bkd_nuniq(self)

Conta o número de dados únicos por variável, desconsiderando valores NaN

  • :return: numero de valores únicos por variável
  • :rtype: pandas.Series
class BkdEda.bkd_desc(self)

Realiza uma descrição estatística dos dados em análise. A descrição apresenta uma variação se os dados fornecidos forem categóricos ou numéricos

  • :return: resumo das principais características dos dados
  • :rtype: pandas.DataFrame ou pandas.Series
class BkdEda.bkd_desc(self)

Fornece as dimensões dos dados em análise (Número de registros x Número de variáveis)

  • :return: dimensão do DataFrame
  • :rtype: tuple
