Módulo de Limpeza

Este é um componente de apoio ao processo de desenvolvimento do Projeto FPDA no período de maio de 2021 à maio de 2023, o qual foi desenvolvido para extração de documentos de código python.

BkdClean

O tratamento de dados será responsável pela limpeza e correção dos dados fornecidos.

Este módulo recebe os dados filtrados pelo módulo de leitura com base nas varáveis escolhidas e faz o tratamento necessário dos dados.

Apresenta as seguintes classes:

BkdClean: Representação dos dados tratados

class BkdClean.__init__(self, bkd_filt, bkd_dict)

Construtor da classe BkdClean

Armazena os dados filtrados e o DataFrame modificado com base no tratamento aplicado.

  • :param bkd_filt: dados filtrados com base nas variáveis escolhidas
  • :type bkd_filt: pandas.DataFrame
  • :param bkd_dict: dicionário contendo escolhas do usuário para preenchimento de dados faltantes
  • :type bkd_dict: dict
class BkdClean.bkd_bad(self)

Realiza a troca de dados 'Bad' por 'NaN' no DataFrame.

class BkdClean.bkd_float(self)

Altera o tipo de dados do DataFrame para float

class BkdClean.bkd_fillna(self)

Preenche dados faltantes utilizando o método escolhido pelo usuário.Caso o usuário não faça escolhas, será realizado a opção "default" na qual os registros com dados faltantes serão removidos.

Os dados podem ser preenchidos utilizando as seguintes chaves:

  • - Valor médio: "mean"
  • - Valor máximo: "max"
  • - Último valor válido: "back"
  • - Valor válido seguinte: "forw"
