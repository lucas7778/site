Módulo de Outliers

Este é um componente de apoio ao processo de desenvolvimento do Projeto FPDA no período de maio de 2021 à maio de 2023, o qual foi desenvolvido para extração de documentos de código python.

BkdOutlier

BkdOutlier será responsável pela detecção e tratamento dos outliers.

Este módulo recebe um DataFrame com as variáveis selecionadas pelo usuário, já tratado, e realiza a detecção e realiza a detecção e tratamento dos outliers existentes

Apresenta as seguintes classes:

BkdOutlier, BkdKNN, BkdAvgKNN, BkdMedKNN, BkdLOF, BkdIForest.

class BkdOutlier.__init__(self, bkd_dados, bkd_model='None')

Classe pai dos métodos disponíveis para detecção e tratamento de outliers

Construtor da classe BkdOutlier.

  • :param bkd_dados: Dados contendo outliers a serem removidos
  • :type bkd_dados: pandas.DataFrame
class BkdOutlier.bkd_model(self, model)

Possibilita acesso ao atributo bkd_model. Este atributo indicará na classe filha o algoritmo de detecção de outlier utilizado.

  • :return: valor contido em bkd_model
  • :rtype: object
class BkdOutlier.bkd_fit(self)

Realiza o fit do método de tratamento de outlier escolhido.

class BkdOutlier.bkd_predict(self)

Realiza a classificação de quais dados são outliers com base no método para tratamento de outlier escolhido.

class BkdOutlier.bkd_remove(self)

Remove da base os registros classificados como outliers pelo método bkd_predict.
