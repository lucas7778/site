Módulo de Manipulação de dados

Este é um componente de apoio ao processo de desenvolvimento do Projeto FPDA no período de maio de 2021 à maio de 2023, o qual foi desenvolvido para extração de documentos de código python.

bml_dtmn

Módulo responsável por realizar a manipulação dos dados para realização das predições

Classe que engloba funções utilitárias de manipulação dados.

Métodos públicos: bdm_dfgrn, bdm_dfpar, bdm_dfsim, bdm_gdiv, bdm_ftab, bdm_gettr, bdm_ids, bdm_splxy

class bml_dtmn.bdm_gettr(bdm_vars, bdm_scale='minmax', bdm_range(-1,1), bdm_sprep=None, bdm_trnds=None, bdm_scale='minmax')

Gera transformadores conforme opções.

  • :param bdm_vars: Nomes das variáveis.
  • :param bdm_scale: Estratégia de conversão de escala/normalização.
  • :param bdm_range: Intervalo da conversão de escala.
  • :param bdm_sprep: Opções especiais de pré-processamento.
  • :param bdm_trnds: Partição de treinamento do dataset.
  • :return: Tupla contendo transformadores e nomes para colunas.
class bml_dtmn.bdm_ids(bdm_dsraw, bdm_cvar, select_cols=True, remove_bad=True, treat_type=True, treat_time=True)

Formata informações temporais, seleciona colunas (variáveis de interesse). Opcionalmente remove linhas com valores 'Bad', troca virgula por ponto e coloca as variáveis em float64.

  • :param bdm_dsraw: DataFrame.
  • :param bdm_cvar: Colunas de interesse.
  • :return: Tupla contendo o DataFrame e sua descrição.
class bml_Dtmn.bdm_splxy(bdm_data, bdm_inn, bdm_outn)

Recebe um dataframe e os nomes dos inputs e output e os separa o dataframe em x e y apropriados.

  • :param bdm_data: DataFrame.
  • :param bdm_inn: Nomes dos Inputs.
  • :param bdm_outn: Nome do Output.
  • :return: Tupla (x,y)
class bml_Dtmn.bdm_dfpar(bdm_df, bdm_dsspl=(0.6,0.2,0.2), bdm_type='simple', bdm_dint=(2,100))

Particiona o DataFrame conforme o método requisitado.

  • :param bdm_df: DataFrame.
  • :param bdm_dsspl: Percentuais das partições.
  • :param bdm_type: Estratégia de particionamento.
  • :param bdm_dint: Intervalo de busca do melhor intervalo de divisão.
  • :return: Tupla contendo os DataFrames das partições.
class bml_Dtmn.bdm_dfsim(bdm_df, bdm_dsspl=(0.6,0.2,0.2))

Particiona o DataFrame de forma simples.

  • :param bdm_df: DataFrame.
  • :param bdm_dsspl: Percentuais das partições.
  • :return: Tupla contendo os DataFrames das partições.
class bml_Dtmn.bdm_dfgrn(bdm_df, bdm_dsspl=(0.6,0.2,0.2), bdm_dint=(2,100))

Particiona o DataFrame de forma granular.

  • :param bdm_df: DataFrame.
  • :param bdm_dsspl: Percentuais das partições.
  • :param bdm_dint: Intervalo de busca do melhor intervalo de divisão.
  • :return: Tupla contendo os DataFrames das partições.
class bml_Dtmn.bdm_gdiv(bdm_dflen, bdm_dsspl, bdm_dint)

Retorna o menor divisor de um número cujas divisões pelas porcentagens passadas seja inteira dentro do intervalo.

  • :param bdm_dflen: Número.
  • :param bdm_dsspl: Percentuais das partições.
  • :param bdm_dint: Intervalo de busca do melhor intervalo de divisão.
  • :return: Divisor
class bml_Dtmn.bdm_ftab(bdm_df, bdm_info)

Formata um objeto DataFrame e sua descrição (pd.DataFrame().describe()) para apresentação, retornando ambos formatados.

  • :param bdm_df: DataFrame.
  • :param bdm_info: DataFrame equivalente a data.describe().
  • :return: Tupla "(bdm_df, bdm_info)" contendo os DataFrames devidamente formatados.
class bml_Dtmn.bdm_mlsk(bdm_tag: str, bdm_inv: bool=False)

Retorna um dicionário que traduz a nomenclatura de hiperparâmetros do pacote para a nomenclatura do scikit-learn.

class bml_Dtmn.bdm_tdict(bdm_mtype: str, bdm_dict: dict, bdm_inv: bool=False)

Traduz um dicionário de parâmetros.

class bml_Dtmn.bdm_fdflt(bdmm_f)
