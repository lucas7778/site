bml

Módulo do Machine Learning bml_mman
Apresenta as seguintes classes:

bmm_Dflt,bmm_MM

Classe bmm_MM, cuja funções principais são oferecer funcionalidades para atender os casos de uso e gerenciar a criação, carregamento, serialização e comparação de modelos.

class bmm_MM.bmm_save(bmm_model, bmm_path, **kwargs)

Salva o modelo no caminho especificado

class bmm_MM.bmm_load(bmm_path, **kwargs)

Carrega o modelo no caminho especificado.

class bmm_MM.bmm_amdls(**kwargs)

Retorna uma tupla contendo os nomes dos modelos disponíveis.

class bmm_MM.bmm_getm(bmm_tag='Multivariate Regression' , bmm_tag=None, bmm_inn=None, bdmm_outn=None, bmm_build=None, bmm_pdata=None, bmm_fit=None, **kwargs)

Retorna o modelo especificado a partir da tag passada. Caso bmm_build seja igual a True e as variáveis bmm_pdata e bmm_fit não sejam passadas, acusará erro.

  • :param bmm_tag: Tag referente ao modelo desejado.
  • :param bmm_name: Nome do modelo a ser criado.
  • :param bmm_inn: Tupla contendo o nome das variáveis de entrada.
  • :param bmm_outn: Tupla contendo o nome da variável de saída.
  • :param bmm_build: True se o modelo já deve ser preparado para uso.
  • :param bmm_pdata: Dicionário contendo as entradas do método bbb_pdata.
  • :param bmm_fit: Dicionário contendo as entradas do método bm_fit.
  • :return: Modelo.
class bmm_MM.bmm_comp(bmm_list, bmm_metrc='r2', bmm_stat='50%', **kwargs)

A partir de uma lista de modelos passada, retorna o melhor segundo a métrica escolhida.

  • :param bmm_list: Lista contendo os modelos a serem comparados.
  • :param bmm_metrc: Métrica escolhida.
  • :param bmm_stat: Estatística a respeito da métrica (não utilizado se bmm_metrc == 'r2').
  • :return: Modelo escolhido.
class bmm_MM.bmm_compt(bmm_list, bmm_metrc='r2', **kwargs)

Retorna um DataFrame apresentado a métrica escolhida para cada modelo da lista passada.

  • :param bmm_list: Lista contendo os modelos a serem comparados.
  • :param bmm_metrc: Métrica escolhida.
  • :return: DataFrame.
class bmm_MM.bmm_auto(bmm_inn=None, bmm_outn=None,bmm_avail=(), bmm_dsraw=None, bmm_setup=None, bmm_score='r2', bmm_grids=None, bmm_metrc='r2', bmm_stat='50%', bmm_name='',**kwargs)

Aplica a funcionalidade de Grid Search a diversos modelos e retorna o melhor.

  • :param bmm_inn: Tupla contendo os inputs.
  • :param bmm_outn: Tupla contendo o output.
  • :param bmm_avail: Tupla contendo tags dos modelos a serem testados.Caso seja igual a "all", todos os modelos disponíveis serão testados
  • :param bmm_dsraw: DataFrame referente a base de dados.
  • :param bmm_setup: Dicionário contendo o setup de pré-processamento.
  • :param bmm_metrc: Métrica utilizada para comparar os modelos.
  • :param bmm_grids:
  • :return: Tupla contendo
class bmm_Dflt.bdm_compt(bmm_f, bmm_obj=None, **kwargs)

Retorna um dicionário com os valores default do método.

  • :param bmm_f: Método
  • :param bmm_obj: Objeto sobre o qual se chama o método.
  • :return: Dicionário com os parâmetros default do método.
class bmm_Dflt.__bmm_gobja(bmm_obj)
