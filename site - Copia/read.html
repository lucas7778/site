Módulo de Leitura

Este é um componente de apoio ao processo de desenvolvimento do Projeto FPDA no período de maio de 2021 à maio de 2023, o qual foi desenvolvido para extração de documentos de código python.

BkdInput

Leitura será responsável pelo tratamento inicial do arquivo de dados

Este módulo realiza a leitura do arquivo de dados fornecidos e faz a seleção das variáveis escolhidas, retornando um DataFrame filtrado com base em tais escolhas.

Apresenta a seguinte classe: BkdInput

class BkdInput.__init__(self, bkd_dict)

Construtor da classe BkdInput

  • :param bkd_dict: informações para manipulação dos dados lidos
  • :type bkd_dict: dict
class BkdInput.bkd_dados(self)

Possibilita acesso aos dados lidos.

  • :return: dados lidos e armazenados
  • :rtype: pandas.DataFrame
class BkdInput.bkd_dict(self)

Possibilita acesso ao dicionário com informações para manipulação.

  • :return: informações para manipulação dos dados lidos
  • :rtype: dict
class BkdInput.bkd_ftype(self)

Checa se o arquivo de dados fornecido existe e possui extensão CSV.

  • :return: mensagem de erro caso exceção seja levantada
  • :rtype: str
class BkdInput.bkd_read(self)

Realiza leitura de um arquivo de dados csv indicado no dicionário bkd_dict.

class BkdInput.bkd_varcheck(self)

Checa se as variáveis escolhidas existem no DataFrame fornecido. Caso o usuário não faça seleção de variáveis, todas serão escolhidas e checagem não ocorre.

  • :return: mensagem de erro caso exceção seja levantada
  • :rtype: str
class BkdInput.bkd_selec(self)

Filtra o DataFrame mantendo as variáveis selecionadas. Além da série temporal, as variáveis que serão mantidas estão presente no bkd_dict. Caso o usuário não selecione variáveis, todas serão mantidas no DataFrame.

class BkdInput.bkd_seltime(self)

Filtra o DataFrame com base no horizonte de estudo escolhido. Se o usuário não fizer escolhas, o horizonte de dados completo será utilizado.
