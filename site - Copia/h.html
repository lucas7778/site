Escopo do projeto

  • Gerar conhecimento em ferramentas analíticas para sistemas elétricos e de processo das plataformas offshore;
  • Gerar base de dados de fatores de carga e de intermitência com base em cenários reais de operação;
  • Padronizar a infraestrutura necessária para aquisição de dados do sistema elétrico;
  • Especificar a topologia mínima de sinais do sistema elétrico que devem ser coletados pelos sistemas supervisórios e de informação;
  • O sistema permitirá a simulação baseada em dados permitindo a avaliação do desempenho energético de FPSOs em modo standalone;
    • Apresentar um roadmap de solução corporativa integrada para aplicação futura;
  • Apoiar o planejamento: sistema baseado em dados para apoiar o dimensionamento de projetos elétricos de FPSOs;
  • Gerar relatórios de apoio (ex.: fator de carga x regime operacional, além de outros que venham a ser mapeados ao longo do processo de desenvolvimento).

Resumo

A tendência de digitalização presente em todos os seguimentos da sociedade, que, sob o ponto de vista da indústria, tem sido denominado de indústria 4.0, dá origem a uma vastidão de dados sobre os mais variados processos. A extração de conhecimento a partir dos dados disponíveis é uma das áreas mais importantes da Inteligência Artificial (IA), incluindo técnicas de Redes Neurais Artificiais e, de forma geral, técnicas de Machine Learning. De fato, a aplicação de técnicas estatísticas clássicas para análise de dados é um campo de atuação antigo e bem fundamentado, muito em voga atualmente em função da vastidão de dados e popularmente conhecido como Analytics. Contudo, a disponibilidade de dados aliada ao crescimento exponencial da capacidade de processamento permite a aplicação também de técnicas avançadas de Inteligência Artificial, no que vem sendo denominado Intelligent Analytics. Quanto mais conhecimento se tem do histórico operacional das plataformas, as estimativas computacionais estarão mais próximas da realidade, proporcionando ganhos, tais como: equipamentos de distribuição e geração otimizados em tamanho, peso e custo; redução de retrabalhos de projeto devido a subdimensionamento proporcionando maior eficiência energética da plataforma. A indústria de óleo e gás está inserida neste contexto de digitalização, dando origem a algumas propostas baseadas em Inteligência Artificial para apoio a tomada de decisão. Apesar da tendência de crescimento da aplicação de técnicas de IA para tratamento de problemas da indústria de petróleo, a literatura ainda é escassa em termos de aplicações destas tecnologias para modelagem baseada em dados de processos de produção. Especificamente sobre insumos energéticos em plataformas de petróleo, não são encontrados trabalhos propondo metodologias para construção de simuladores baseados em dados de processos produtivos em plataformas. Da mesma forma, em relação ao projeto elétrico de plataformas, a construção de tais ativos considera parâmetros usuais de projeto que contemplam medidas conservadoras de risco. Contudo, registros de produção mostram que os equipamentos elétricos destas plataformas atuam em condições de carregamento muito inferiores àquelas de projeto, permitindo eventuais ampliações da produção da planta sem adição de riscos em termos de sobrecarga destes equipamentos. Analogamente, quanto mais exatas forem as estimativas dos fatores de carga e de intermitência dos equipamentos, maior é a economia no tamanho, peso e custo dos equipamentos de distribuição e geração nos casos de sobredimensionamento e menor é o risco de alterações de projeto no caso de subdimensionamento.

Ambiente de desenvolvimento
  • A linguagem de programação definida para este projeto é a linguagem Python 3.9.5 Python.
  • O ambiente de produção será direcionado para a plataforma Windows 10, requisito do cliente, portanto, todos os testes durante o ciclo de vida de desenvolvimento do sistema computacional levará em conta esse fato, isso será considerado fator crítico de sucesso.
  • O projeto será desenvolvido segundo o modelo incremental e iterativo, trabalhando com um misto de metodologia convencional com base nas melhores práticas de mercado e referenciadas pelo PMBOK 6 e metodologia Ágil notadamente o SCRUM.
  • Toda a documentação do projeto se encontra no github e as codificações no gitlab Petrobras.
