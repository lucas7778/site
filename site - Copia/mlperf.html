bml_Perfm

class bml_Perfm.bp_gets(bp_ypred, bp_ytrue)

Toma métricas de uma predição, em relação a uma ground truth.

A partir de tags passadas retorna o valor da métrica pertinente.

  • :param bp_model: Modelo.
  • :param bp_metrc: Métrica desejada.
  • :param bp_stat: Estatística sobre a métrica.
  • :return: Valor da métrica.
class bml_Perfm.bp_apred(bp_bbox, bp_df='val')

Retorna métricas do modelo.

  • :param bp_bbox: Modelo
  • :param bp_df: "trn", "val", "tst" ou DataFrame com variáveis a serem utilizadas para comparação (deve conter valores para todas as variáveis utilizadas pelo modelo (input e output).
  • :return: Tupla (bp_sdf, bp_smry, bp_r2), onde bp_sdf é um DataFrame comparando as predições do modelo com os valores de referencia encontrados em bp_df, bp_smry é um DataFrame contendo estatísticas descritivas de bp_sdf e bp_r2 é o coeficiente de determinação.
