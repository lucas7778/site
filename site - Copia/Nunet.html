Módulo de Modelagem de novas redes

Este é um componente de apoio ao processo de desenvolvimento do Projeto FPDA no período de maio de 2021 à maio de 2023, o qual foi desenvolvido para extração de documentos de código python.

NuNet

Esta classe é utilizada para construir uma nova rede elétrica.
class NuNet.__init__(self, bpf_data=None)

Essa função inicializa objetos da classe NuNet. Ela requer apenas 1 parâmetro de entrada, sendo aqui denominado bpf_data.

bpf_data é, obrigatoriamente, um dicionário Python. Até a presente versão do código, suas possíveis chaves são:
  • 'bpf_info'* ---> armazena dados básicos da rede
  • 'bpf_bus'* ---> armazena os dados das barras
  • 'bpf_line' ---> armazena os dados das linhas
  • 'bpf_tr' ---> armazena os dados dos trafos de 2 enrol.
  • 'bpf_tr3' ---> armazena os dados dos trafos de 3 enrol.
  • 'bpf_swt' ---> armazena os dados de chaves seccionadoras
  • 'bpf_load'* ---> armazena os dados das cargas
  • 'bpf_gen' ---> armazena os dados dos geradores (PV)
  • 'bpf_extrn'* ---> armazena os dados da rede externa (swing)
  • ' 'bpf_shunt' ---> armazena os dados de elementos shunt
  • ' OBS: o asterisco (*) indica que a presença da chave é obrigatoria.
Cada chave de bpf_data também armazena um dicionário. A descrição detalhada é dada a seguir:
  • 'bpf_info': { 'bpf_name': (string), 'bpf_fhz': (float), 'bpf_snmva': (float) }
  • 'bpf_bus': { 'bpf_name': (string list), 'bpf_vnkv': (float list), 'bpf_on': (boolean list) }
  • 'bpf_line': { 'bpf_name': (string list), 'bpf_frbus': (string list), 'bpf_tobus': (string list), 'bpf_lenkm': (float list), 'bpf_rpkm': (float list), 'bpf_xpkm': (float list), 'bpf_cpkm': (float list), 'bpf_gpkm': (float list), 'bpf_maxka': (float list), 'bpf_prll': (integer list), 'bpf_dfact': (float list), 'bpf_on': (boolean list) }
  • 'bpf_tr': { 'bpf_name': (string list), 'bpf_hvbus': (string list), 'bpf_lvbus': (string list), 'bpf_snmva': (float list), 'bpf_vnhv': (float list), 'bpf_vnlv': (float list), 'bpf_vkpc': (float list), 'bpf_vkrpc': (float list), 'bpf_pfekw': (float list), 'bpf_iopc': (float list), 'bpf_shift': <ângulo de defasagem do trafo em graus> (float list), 'bpf_on': (boolean list) }
  • 'bpf_tr3': { 'bpf_name': (string list) 'bpf_hvbus': (string list) 'bpf_mvbus': (string list) 'bpf_lvbus': (string list) 'bpf_vnhv': (float list) 'bpf_vnmv': (float list) 'bpf_vnlv': (float list) 'bpf_snhv': (float list) 'bpf_snmv': (float list) 'bpf_snlv': (float list) 'bpf_vhmp': (float list) 'bpf_vmlp': (float list) 'bpf_vhlp': (float list) 'bpf_rvhmp': (float list), 'bpf_rvmlp': (float list), 'bpf_rvhlp': , (float list), 'bpf_pfekw': , (float list), 'bpf_iopc': (float list), 'bpf_on': (boolean list) }
  • 'bpf_load': { 'bpf_name': (string list), 'bpf_busnm': (string list) 'bpf_pmw': , (float list), 'bpf_qmvar': (float list) 'bpf_cnstz': (float list), 'bpf_cnsti': (float list), 'bpf_scale': (float list), 'bpf_on': (boolean list) }
  • 'bpf_gen': { 'bpf_name': (string list), 'bpf_busnm': (string list), 'bpf_pmw': (float list), 'bpf_vpu': (float list), 'bpf_scale': (float list) }
  • 'bpf_extrn': { 'bpf_name': (string), 'bpf_busnm': (string), 'bpf_on': , (boolean), 'bpf_vmpu': (float), 'bpf_vadeg': <ângulo de fase em graus> (float) }
  • 'bpf_shunt': { 'bpf_name': (string list), 'bpf_busnm': (string list), 'bpf_qmvar': (float list), 'bpf_vnkv': (float list), 'bpf_step': (integer list), 'bpf_on': (boolean list) }
  • 'bpf_swt': { 'bpf_name': (string list) 'bpf_busnm': (string list) 'bpf_elety': (string list) "b", no caso de conectar barra-barra "l", no caso de conectar barra-linha "t", no caso de conectar barra-trafo de 2 enrol. "t3", no caso de conectar barra-trafo de 3 enrol., 'bpf_elenm': (string list) }

O atributo self.bpf_data é inicializado com o parâmetro de entrada descrito acima (isto é, bpf_data).

Já o atributo self.bpf_net é inicializado como None. Ele será modificado quando o método bpf_build() for utilizado.
